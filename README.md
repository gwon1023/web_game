# Beat Breaker

A short rhythm-action browser game MVP built with Vite, TypeScript, and Phaser 3.

## Controls

- `J`: Light Attack
- `K`: Heavy Attack
- `Space`: Finisher

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project includes a GitHub Pages workflow. Pushes to `main` build the Vite app and publish `dist`.
